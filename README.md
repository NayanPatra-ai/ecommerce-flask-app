# 🛒 E-Commerce Shopping Cart (Flask)

🚀 A fully functional E-Commerce web application built using *Python Flask*.  
This project demonstrates real-world backend logic, user authentication, and dynamic UI rendering using Flask and Jinja templates.

## ✨ Features

- 🛍️ Product listing with images  
- ➕ Add to Cart functionality  
- ❌ Remove items from cart  
- 💳 Checkout system  
- 🔐 User Login & Registration  
- 🎨 Clean UI with CSS styling  
- ⚡ Dynamic rendering using Flask templates  

---

## 🛠️ Tech Stack

- *Backend:* Python (Flask)  
- *Frontend:* HTML, CSS  
- *Templating Engine:* Jinja2  
- *State Management:* Flask Sessions
